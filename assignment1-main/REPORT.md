# Laporan Programming Assignment 1: Basic C++
Program ini berfungsi untuk menghitung usia seseorang dalam satuan tahun dan bulan serta menentukan hari kelahiran melalui input tanggal berformat DD/MM/YYYY. Kode ini menggunakan library ctime untuk mengolah waktu sistem melalui struktur data tm dan fungsi localtime guna mendapatkan tanggal saat ini, serta memanfaatkan fungsi mktime untuk menentukan hari (seperti Senin atau Selasa) secara otomatis berdasarkan indeks mingguan nol sampai enam. Dalam prosesnya, program memakai sstream atau stringstream untuk membedah teks input menjadi variabel angka terpisah, sementara logika pada fungsi yearsOld dan monthsOld menerapkan perhitungan selisih tahun dan bulan yang dilengkapi dengan pengecekan kondisi atau if-statement untuk memastikan hasil tetap akurat jika tanggal sekarang belum melewati tanggal lahir. Selain itu, kode ini menerapkan manajemen memori manual dengan operator new untuk menyiapkan objek dan delete untuk menghapusnya guna mencegah kebocoran memori, serta menggunakan array string untuk menerjemahkan angka indeks hari ke dalam nama hari dalam bahasa Indonesia.
contoh input:20/20/2000
contoh output:25 295 Senin
