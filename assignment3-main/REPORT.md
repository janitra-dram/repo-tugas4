Pengerjaan tugas ini difokuskan pada implementasi pemrograman berorientasi objek untuk mengklasifikasikan jenis segitiga berdasarkan koordinat tiga dimensi. Struktur kode dibagi ke dalam beberapa modul utama, yaitu Point2D untuk merepresentasikan titik koordinat dan Triangle yang memproses logika geometri. Dalam proses pengembangannya, saya menggunakan pengulangan untuk menerima input sebanyak 3 * N titik sesuai instruksi yang diberikan, di mana setiap tiga titik dikelompokkan menjadi satu objek segitiga yang kemudian disimpan dalam sebuah vektor.
Tantangan utama dalam pengerjaan ini adalah memastikan keakuratan perhitungan jarak antar titik menggunakan rumus Euclidean serta menentukan ambang batas toleransi untuk pengecekan kondisi siku-siku. Karena perhitungan menggunakan tipe data float sering kali menghasilkan selisih angka desimal yang kecil, saya menerapkan fungsi absolut dengan toleransi tertentu agar identifikasi segitiga siku-siku tetap akurat meskipun ada pembulatan nilai. Program ini secara otomatis akan mengklasifikasikan input ke dalam kategori sama sisi, sama kaki, siku-siku, atau sembarang melalui fungsi pengecekan yang terintegrasi.
Seluruh proses kompilasi dilakukan pada lingkungan WSL Ubuntu menggunakan Makefile untuk mengotomatisasi pembuatan file objek di dalam direktori obj. Pengaturan jalur file header (.hpp) dan implementasi (.cpp) disesuaikan dengan struktur folder proyek agar proses linking antar modul berjalan lancar. Hasil akhirnya adalah sebuah file eksekusi yang mampu memproses input secara massal dan memberikan output berupa label jenis segitiga pada setiap barisnya sesuai dengan urutan input yang diterima.
